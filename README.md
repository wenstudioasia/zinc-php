# zinc-php

PHP SDK for [ZincSearch](https://docs.zincsearch.com/).

This package was first developed for private usage.

Development environment:

- OS : Arch Linux
- PHP: v7.2
- Composer: v2.4.1
- ZincSearch server: v0.3.1
- Editor: VSCode

## Install

```shell
composer install wenstudioasia/zinc-php
```
