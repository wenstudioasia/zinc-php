# zinc-php

PHP SDK for [ZincSearch](https://docs.zincsearch.com/).

This package was developed for private usage.

Development environment:

- OS : Arch Linux
- PHP: v7.2
- composer v2.4.1
- ZincSearch server: v0.3.1
- Editor: VSCode

## install

```shell
composer install wenstudioasia/zinc-php
```
